🤖 Chat Embedado com Parlant

Sobre este exemplo

Este é um exemplo de como integrar o chat do Parlant em seu site. O chat pode ser usado de duas formas:

1. Chat Inline (ao lado)

O chat aparece diretamente na página, integrado ao layout do seu site.

2. Chat Flutuante (botão no canto)

Um botão flutuante que abre o chat em uma janela modal. Clique no botão verde no canto inferior direito para testar.

Recursos do Chat:

  • Interface moderna e responsiva
  • Integração com API do Parlant
  • Suporte a múltiplas sessões
  • Histórico de conversas
  • Agentes especializados (como o Dr. CLT)

Como integrar:

  1. Certifique-se que o servidor Parlant está rodando
  2. Configure a URL base da API
  3. Inclua o iframe ou componente React
  4. Personalize o estilo conforme necessário
